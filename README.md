# Digital Library 📚

A Flutter-based digital library app for borrowing and buying books with an XP and leveling system.

---

## 🚀 Features

- 📖 Borrow books (7, 14, 30 days)
- 💰 Buy premium books
- 🎮 XP and leveling system
- 🏆 Achievements and badges
- 🎨 Book categories:
  - Kids 🧸
  - Horror 👻
  - Romance 💖
  - Drama 🎭
  - Fantasy 🐉
- 👤 Personal profile with statistics
- 🖼️ Custom avatar selection
